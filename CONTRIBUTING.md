# 贡献指南 / Contributing Guide

感谢您对 waqupin 项目的兴趣！这是一个AI小说写作插件项目。

## 项目结构 / Project Structure

```
waqupin/
├── src/                 # 源代码目录
│   ├── core/           # 核心功能模块
│   ├── plugins/        # 插件扩展
│   ├── utils/          # 工具函数
│   └── config/         # 配置文件
├── docs/               # 文档目录
├── examples/           # 示例代码
├── tests/              # 测试代码
└── README.md           # 项目说明
```

## 如何贡献 / How to Contribute

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

## 开发指南 / Development Guidelines

### 代码风格 / Code Style
- 使用清晰的变量和函数命名
- 添加必要的注释，特别是复杂逻辑部分
- 保持代码简洁和可读性

### 提交信息 / Commit Messages
- 使用清晰简洁的提交信息
- 如果可能，使用中英文双语说明

### 测试 / Testing
- 在提交前确保代码能够正常运行
- 为新功能添加适当的测试用例

## 问题反馈 / Issue Reporting

如果您发现任何问题或有改进建议：
1. 检查是否已经有类似的 Issue
2. 创建新的 Issue，提供详细的问题描述
3. 如果可能，提供复现步骤
