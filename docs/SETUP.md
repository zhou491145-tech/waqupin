# Waqupin 项目设置指南

## 项目简介

Waqupin 是一个AI小说写作插件，旨在帮助AI在写作长篇小说时保持记忆和连贯性，避免偏离主线和产生幻想。

## 核心功能目标

1. **记忆管理**: 支持百万字级别的小说创作记忆
2. **约束规则**: 实现多种限制规则，确保AI按要求写作
3. **提示词管理**: 支持自定义提示词，控制写作习惯和用词
4. **仿写功能**: 可以读取参考小说内容，实现仿写功能

## 项目结构说明

### /src 目录
项目的主要源代码目录：

- **core/**: 核心功能模块
  - 记忆管理系统
  - AI对话接口
  - 小说内容存储和检索

- **plugins/**: 插件扩展
  - 不同AI模型的适配器
  - 自定义功能扩展

- **utils/**: 工具函数
  - 文本处理工具
  - 配置读取工具
  - 日志工具

- **config/**: 配置文件
  - 默认配置模板
  - 提示词模板
  - 规则配置

### /docs 目录
项目文档：
- API文档
- 使用指南
- 开发文档

### /examples 目录
示例代码和配置：
- 基础使用示例
- 高级功能示例
- 配置文件示例

### /tests 目录
测试代码：
- 单元测试
- 集成测试
- 端到端测试

根据项目需求，建议使用以下技术：
